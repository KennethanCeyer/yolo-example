# yolo-example
An image recognition with yolo example written by PyTorch

---

## Getting Started

### Installation

1. pipenv

```bash
$ pip install pipenv
```

1-1. pytorch

> If you're already installed pytorch, Skip this step 

```bash
```

2. clone repository

```bash
$ git clone https://github.com/KennethanCeyer/yolo-example.git
$ cd yolo-example
```

3. install all dependencies

```bash
$ pipenv install
```
